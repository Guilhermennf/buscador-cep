# Requisitos para a aplicação

É necessários seguir todos os passos abaixo para que a aplicação funcione

### Primeiro Passo

Criar uma pasta para este projeto e clonar o repositório

```
mkdir projeto
cd projeto
git clone  .
```

### Segundo passo

Instalar as dependências

```
yarn
```

### Terceiro passo

Iniciar a aplicação

```
yarn start
```
